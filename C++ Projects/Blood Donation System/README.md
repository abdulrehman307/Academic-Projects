# Blood Donation System (C++)

This is a console-based Blood Donation System developed as an academic project during my Computer Science studies.

The main goal of this project was to practice C++ fundamentals and file handling concepts.

## Features
- Add donor records
- Add recipients Record
- Stores Donation Records
- Display Donors List
- View stored donor data
- Search function for searching blood available
- Data stored using file handling

## Technologies Used
- C++
- File Handling
- Command Line Interface (CLI)

## Learning Outcome
- Improved understanding of file operations in C++
- Better use of conditional statements and loops
- Experience in building a complete working console program

## Limitations
- No graphical user interface
- Basic validation only

This project was created for learning purposes as part of my academic coursework.
